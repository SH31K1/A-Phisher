# 🛡️ A-Phisher – Phishing Awareness Tool

## 🔥 Overview
A cybersecurity awareness tool designed to simulate phishing attacks and analyze user behavior in a controlled environment.

---

## ⚙️ Features
- 35+ phishing templates
- Simulated credential capture (safe environment)
- User interaction tracking
- OWASP Top 10 vulnerability mapping

---

## 🛠 Tools Used
- Burp Suite
- SQLMap
- Gobuster
- Kali Linux

---

## 📊 Results
- Analyzed user response patterns to phishing attacks
- Demonstrated common security awareness gaps
- Improved understanding of social engineering risks
